# Clearweb

The web, without the AI.

Clearweb is a free, open-source browser extension that silently removes AI-generated content and chatbot widgets from every website you visit. No account. No data collected.


## Install right now (takes 2 minutes)

Chrome, Edge or Brave:

1. Go to github.com/jake6214-hue/clearweb
2. Tap the green Code button and select Download ZIP
3. Unzip the folder on your computer
4. Open Chrome and go to chrome://extensions
5. Turn on Developer mode using the toggle in the top right
6. Click Load unpacked and select the unzipped folder
7. The Clearweb icon appears in your toolbar. Done.

Firefox:

1. Download and unzip the folder as above
2. Go to about:debugging in Firefox
3. Click This Firefox, then Load Temporary Add-on
4. Select the manifest.json file inside the folder

Chrome Web Store and Firefox Add-ons submissions coming soon for one-click install.


## What it removes

- Google AI Overviews
- Bing Copilot sidebar
- Gemini panels
- Reddit, YouTube and Amazon AI summaries
- Stack Overflow GPT answers
- Floating chatbot widgets (Intercom, Drift, Tidio, Zendesk, HubSpot and more)
- Any element labelled "AI-generated" or "Powered by AI"


## How it works

Runs silently on every page using four detection methods: site-specific selectors, chatbot widget IDs, class/ID pattern matching, and keyword scanning. Elements are hidden not deleted, so you can toggle it off any time to restore the page instantly.


## Privacy

Zero data collection. No analytics. No telemetry. Runs entirely in your browser.
